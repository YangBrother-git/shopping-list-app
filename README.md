# 쇼핑 리스트 앱

로컬 스토리지 기반의 한국어 쇼핑 리스트 웹 앱입니다.

## 실행 방법

`shopping-list.html` 파일을 브라우저에서 직접 열면 됩니다. 별도 서버 불필요.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 완료/미완료 체크박스
- 전체 / 미완료 / 완료 필터
- 완료 항목 일괄 삭제
- localStorage로 새로고침 후에도 데이터 유지

## 테스트

```bash
npm install
npm test
```

Playwright 기반으로 7가지 시나리오를 자동 테스트합니다.
